Fluid Artist

Is a form of abstract art that uses acrylic paints with a runny (fluid)consistency. The acrylic paints react with each other when combined together to make interesting and visually organic motifs. This type of art is fun for all ages. Fluid acrylics can be used on many types of substrates and in many different forms such as pouring, dripping, swirling, glazing, dipping and many other effects.

Web Developer

A web developer is a programmer who specializes in, or is specifically engaged in, the development of World Wide Web applications using a client–server model. The applications typically use HTML, CSS and JavaScript in the client, PHP, ASP.NET (C#) or Java in the server, and http for communications between client and server. A web content management system is often used to develop and maintain web applications.
